Technical Specifications

System Requirements

  • Windows: Windows 10/11 (64-bit)
  • Android: Android 6.0+ (API level 23+)
  • RAM: 4GB minimum, 8GB recommended
  • Network: WiFi connection required

Network & Security

  • Protocol: WebSocket (port 8080)
  • Discovery: UDP broadcast (port 41234)
  • Security: Local network only
  • Trust System: Device-based authentication

Features

  • Sensitivity: 0.1x to 10x adjustable
  • Gestures: Multi-touch support
  • Latency: Ultra-low (<50ms)
  • Precision: Pixel-perfect control
